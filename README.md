 # 🛡️ AI Security Dashboard

An AI-powered cybersecurity dashboard for phishing URL risk analysis built using Python and Streamlit.

## 🚀 Live Demo

https://ai-security-dashboard-3ipwqmhh8aoqrlp6gisxcl.streamlit.app/

## 📌 Project Overview

The AI Security Dashboard analyzes URLs and identifies potential phishing threats using feature extraction and risk scoring techniques. It provides an interactive web interface for users to assess URL security and visualize threat statistics.

## ✨ Features

* URL Risk Analysis
* Phishing URL Detection
* Feature Extraction Engine
* Risk Score Calculation
* Analysis History Tracking
* Threat Detection Statistics
* Safe vs Threat Visualization (Pie Chart)
* CSV Security Report Download
* Interactive Streamlit Dashboard

## 🛠️ Technologies Used

* Python
* Streamlit
* Pandas
* Plotly
* Scikit-Learn
* NumPy
* Joblib

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/pragati5008/AI-Security-Dashboard.git
cd AI-Security-Dashboard
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```
